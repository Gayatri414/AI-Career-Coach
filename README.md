🚀 AI Career Coach

An AI-powered career guidance platform built using modern web technologies to help users create ATS-friendly resumes, improve skills, prepare for interviews, and track career growth.
✨ Features
🤖 AI-based career recommendations
📄 ATS-Friendly Resume Builder
🧠 Technical & Aptitude Quizzes
📊 Performance Analytics Dashboard
🔐 Secure Authentication
📱 Fully Responsive UI
⚡ Smooth Animations & Modern Design
Frontend: Next.js, React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT
AI Integration: Gemini API

Deployment:ai-career-coach-6lqrdlenm-gayatri-gades-projects.vercel.app
